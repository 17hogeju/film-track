# FilmTrack
FilmTrack is a mobile application that helps you track your media interests, discover new media, and identify which subscription services are right for you.

## Abstract
FilmTrack provides a simple system for tracking your media interests and identifying which streaming services align most with your media preferences. It utilizes machine learning to recommend titles based on your media collectionâ€™s metadata and media that similar users enjoyed.
